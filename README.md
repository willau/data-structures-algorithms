## Sort
#### Complexity
+ Merge sort
    + Worst   --> O(n.log(n))
    + Average --> O(n.log(n))
+ Heap sort
    + Worst   --> O(n.log(n))
    + Average --> O(n.log(n))
+ BST sort
    + Worst   --> O(n^2)
    + Average --> O(n.log(n))
+ Quick sort 
    + Worst   --> O(n^2)
    + Average --> O(n.log(n))
+ Insertion sort
    + Worst   --> O(n^2)
    + Average --> O(n^2)
+ Selection sort
    + Worst   --> O(n^2)
    + Average --> O(n^2)
